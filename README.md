Zendesk Help Desk Ticket Automation
Overview

This project demonstrates a simulated help desk workflow using Zendesk along with a small API automation tool used to automatically generate support tickets. The goal of this project was to practice real world IT support operations, including ticket management, troubleshooting, and customer communication.

The automation component generates incoming support tickets, while the help desk portion focuses on manually responding to, troubleshooting, and resolving tickets within Zendesk.

This project highlights skills commonly used in IT Support, Help Desk, and Technical Support roles.

Key Features

• Automated ticket generation using an API
• Manual ticket triage and troubleshooting in Zendesk
• Professional help desk communication with users
• Issue diagnosis and resolution tracking
• Documentation of common technical issues
